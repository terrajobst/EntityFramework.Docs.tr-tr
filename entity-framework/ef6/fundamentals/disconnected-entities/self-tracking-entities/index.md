## <a name="security-considerations"></a><span data-ttu-id="ce654-145">Güvenlik Değerlendirmeleri</span><span class="sxs-lookup"><span data-stu-id="ce654-145">Security Considerations</span></span>  

<span data-ttu-id="ce654-146">Aşağıdaki güvenlik konuları hesaba kendi izleme varlıkları ile çalışırken dikkat edilmelidir:</span><span class="sxs-lookup"><span data-stu-id="ce654-146">The following security considerations should be taken into account when working with self-tracking entities:</span></span>
